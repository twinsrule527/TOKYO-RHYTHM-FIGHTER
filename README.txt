Tokyo Rhythm Fighter Sprint 1


Build Instructions

	The game is in a zip file - when extracted, you can play by running the Application File titled "Tokyo Rhythm Fighter"

Gameplay Instructions

	Space - Make an attack. If on beat, will deal damage to enemy and will block their attack (if they are making one.

	Enemy changes colors:
		blue & green indicate that it is preparing for an attack
		it changes to red the beat before it actually attacks
		black means that it is recovering from having made an attack
		it will flash white at certain points of interaction - when you block an attack, and sometimes at the instance it makes an attack
